#Server

1. Create Database with name customerdb
2. Install Node Modules using command "npm install"
2. To run server use command "node app.js"

#UI

1. Install Node Modules using command "npm install"
2. Use command "npm start" to run project


#Details

1. Form Validation is done using React Bootstrap
2. Email Id is unique(checks while creating new or updating)
3. Data shown in tble is sorted, Last created customer shows on top
4. Searching (name, email, mobile, status, address)
